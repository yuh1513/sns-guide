TYPE 01
ナレーション型
🎙️
スタイル音声と映像を別々で撮るスタイル
目的日々の仕事紹介や普段使う道具の紹介など、紹介系と相性〇
撮り方引いて撮る、寄って撮る、様々な角度で撮ったり、右から左へ流れるように撮ったりすると〇
コツ色々な素材があると◎、ナレーションは声を作らずそのままが〇（動画の内容で元気な声や落ち着いた声など、雰囲気を少し変えると◎）
音
ナレーションの録り方は、動画を撮影して喋る、動画と喋る内容を合わせたい場合は動画を見ながら喋ると◎
TYPE 02
インタビュー型
🎤
スタイル対談スタイル
目的コメント、質問の答えや仕事への想い、利用者様の声を聴くなどリアルな部分を発信
撮り方2人でも1人でも文字だけでもOK、上司や同僚など人と人との対話があると◎、質問者がカメラを持ち撮影するとリアルな感じがして○
コツ内容によって、硬い感じや楽し気な感じなど変えていくとメリハリがつく
音
内容にもよるが毎回、ほぼ同じBGMにすると定番化し見やすくなる
TYPE 03
Vlog型
📹
スタイル密着型、一人の仕事内容を密着して撮影するスタイル
目的同業者や施設利用者の共感や視聴者の興味を得るために。日常系と相性〇
撮り方基本的に一人がその日の仕事内容やルーティーンを撮っていく
コツ自分のお気に入りのアイテム紹介やドアを開ける、靴を履く、窓の外の景色、といった「何気ないカット」があると◎
音
ASMR的要素を含むので、一つ一つの動作の音、会話や人の声を撮れていると◎
TYPE 04
シネマ型
🎬
スタイル会社イメージやイベントなどを撮影するスタイル、プロモーション系と相性〇
目的新しいキャンペーンや広告を打ち出したいときに
撮り方様々な画角や素材が必要、細かいところまで撮影できると◎
コツ少し気取った角度で撮ると雰囲気がでて〇
音
BGMを追加し雰囲気をブースト、ナレーション式にして声を当てても〇
TYPE 05
あるある型
😄
スタイル仕事内容のあるあるなどの共感スタイル、ネタ系と相性〇
目的あるあるや失敗談などを紹介して共感を獲得したり、興味を深め視聴者を増やす
撮り方文字だけでも〇、会話のなかでその内容を切り抜いても〇、基本カメラを固定して撮影したり、インタビュー形式でも〇
コツ内容は必ずポジティブに、自分たちが楽しむことを大事にすると見やすくて〇
音
流行りの音楽とともに撮影したり、ポジティブな雰囲気をだせるBGMがあると◎
TYPE 06
ミーム型
🔥
スタイル流行りを取り入れたスタイル
目的トレンドを取り入れ、誰でも見やすくし、内容の幅を広げる
撮り方流行っている、もしくは流行りそうなネタを取り入れ、自分たちの内容と当てはめ撮影すると◎
コツクオリティを求めると他のポストを差別化を図れるので◎
音
流行りのBGM
TYPE 07
POV型
👁️
スタイル仕事内容を撮影するスタイル
目的仕事内容の紹介、リアルな部分の露出によって忖度のないリアルな動画が撮れる
撮り方一人称視点で撮影していくとリアルさが増して見やすくなる◎
コツ自分が普段見ている景色を意識すると◎
音
内容合わせBGMや、ナレーションを追加し見やすくする
